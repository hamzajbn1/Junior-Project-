# Frontend Dashboard Prototype

This folder contains the frontend prototype for the Research Trend Explorer project.

## Pages
- Overview
- Topics
- Authors
- Organizations
- Explorer

## Current status
- The overview page is the main polished prototype and includes charts and summary sections.
- The remaining pages are structured prototype pages prepared for later implementation.

## Purpose
This frontend structure is designed to support later backend integration with the CSKG and OpenAlex data pipeline.
